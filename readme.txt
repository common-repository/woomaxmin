WooCommerce Minimum & Maximum Purchase Limit plugin gives you the ability to set up minimum and maximum purchase limit for your customers.



== Description ==

WooCommerce Minimum & Maximum Purchase Limit plugin gives you the ability to set up minimum and maximum purchase limit for your customers. If the customer cross the purchase limit or try to buy below minimum purchase limit , the plugin will show a custom warning message to your customers as well as you can also disable add to cart and checkout button if your customers try to buy outside of the limit you set .

WooCommerce Minimum & Maximum Purchase Limit plugin also gives you the ability to set up Purchase Restriction date so that you can keep the limit for a certain period . There is also option to setup limit for life time. You can also put purchase limit base on user role. 

Fully tested with WooCommerce 2.6.9 .

== Installation ==

= WooMaxMin Requirements =

*   WooCommerce 2.5+
*   WordPress 4.1+
*   PHP 5+

= Install Instructions =

1. Upload the folder `woomaxmin` to the `/wp-content/plugins/` directory of your site
2. Activate the plugin through the 'Plugins' menu in WordPress
